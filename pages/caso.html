Caso práctico de aplicación

Un caso práctico de criptografía de clave pública es el uso del algoritmo RSA en la comunicación segura de correos electrónicos. En este caso:
1. El remitente obtiene la clave pública del destinatario.
2. Usa la clave pública para cifrar el contenido del correo
3. Envía el mensaje cifrado.
4. El destinatario usa su clave privada para descifrar el contenido.

Además, en sistemas de correo electrónico seguros como PGP (Pretty Good Privacy) o S/MIME (Secure/Multipurpose Internet Mail Extensions), la criptografía de clave pública también se emplea para la autenticación de remitentes mediante firmas digitales. De esta manera, se garantiza no solo la confidencialidad del mensaje, sino también su integridad y autenticidad.
